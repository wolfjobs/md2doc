快速搜索文档内容，支持键盘快捷键 (Ctrl+K)

📖 自动目录

自动生成文档目录，方便快速导航

📋 代码复制

一键复制代码块，提升开发体验

🚀 访问方式

            
            # 开发模式
            npm run dev
            # 访问: http://localhost:3000

            # 构建生产版本
            npm run build
            # 输出目录: dist/

            # 本地服务器
            npm run serve
            # 访问: http://localhost:8080

📁 文件结构

            
            docs-site/
            ├── src/
            │ ├── data/docs.json # 文档配置
            │ ├── docs/ # Markdown 文档
            │ ├── js/app.js # 主应用逻辑
            │ └── styles/ # 样式文件
            ├── dist/ # 构建输出
            ├── index.html # 主页面
            └── package.json # 项目配置

🛠️ 技术栈

  • 构建工具: Vite
  • Markdown 解析: Marked.js
  • 样式: 原生 CSS + CSS 变量
  • 图标: Font Awesome
  • 字体: Inter

📖 使用说明

文档站点已经构建完成，您可以：

  1. 直接访问 dist/index.html 查看效果
  2. 使用 npm run serve 启动本地服务器
  3. 将 dist 目录部署到任何静态文件服务器
  4. 根据需要修改 src/docs/ 下的 Markdown 文件
